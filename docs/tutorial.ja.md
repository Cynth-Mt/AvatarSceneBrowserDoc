# クイックスタート

---

## ステップ1：スクリーンショットを撮る

インポート後、Sceneウィンドウ上部のツールバーに2つのボタンが表示されます。

![ツールバー](img/toolbar01.png)
![ツールバー](img/toolbar02.png)

スクリーンショットボタンをクリックすると、現在のシーンのサムネイルが撮影されます：

![スクリーンショットボタン（ライトテーマ）](img/gui_Screenshot_light.png){ width="80" }

または上部メニューから手動で更新することもできます：

`Tools → CYN-lab → VRCAvatarSceneBrowser → Main Window`

メインウィンドウで **現在開いているシーンのスクリーンショットを更新** をクリックします。

![現在開いているシーンのスクリーンショットを更新](img/刷新当前打开的场景截图.png)

---

## ステップ2：スクリーンショットライブラリを確認

撮影したシーンの画面が中央エリアに一覧表示されます。

[→ メインウィンドウ詳細](main-window.md)

---

## ステップ3：シーンを開く

シーンを開く方法はいくつかあります：

1. **サムネイルをダブルクリック** — 最も素早い方法
2. **サムネイルを右クリック → シーンを開く**
3. **シーンを選択 → 右側パネルの「シーンを開く」ボタンをクリック**

---

## ステップ4：タグを追加する

シーンを選択した後、右側の詳細パネルでタグを追加できます。

タグの編集・管理については [タグ管理](tag-manager.md) を参照してください。

---

## 複数Avatarシーンの使い方

1つのUnityシーンに複数のVRChat Avatarが含まれている場合、表示状態ごとに個別のスクリーンショットを保存できます：

1. HierarchyでスクリーンショットしたいAvatarだけを表示（他は非表示に）する
2. ツールバーのスクリーンショットボタンをクリック → **Avatar設定スクリーンショット**（緑色ボーダー）として自動保存されます
3. 他のAvatarについても同様の手順を繰り返す

Avatarの状態を復元する：

- 緑色ボーダーのサムネイルをクリックして設定を選択
- 右側の **Avatar 設定** エリアで ▶ ボタンをクリック → AvatarのON/OFFが自動的に切り替わります

!!! info "注意"
    Avatar設定機能はVRChat SDK（VRCSDK3）がある場合に完全に動作します。SDKがない場合でも、スクリーンショット機能は通常どおり使用できます。

## Unityメニュー

```
Tools
└── CYN-lab
    └── VRCAvatarSceneBrowser
        ├── Main Window              # メインウィンドウ
        ├── GUID Batch Update Tool   # GUID一括更新ツール
        └── Screenshot Settings...   # スクリーンショット設定
```

### Main Window（メインウィンドウ）
日常のシーン管理と閲覧。

### GUID Batch Update Tool（GUID一括更新ツール）
シーンファイルの名前変更や移動後にデータの関連付けを修復するためのツール。[→ 詳細](guid-batch-update-tool.md)

### Screenshot Settings...（スクリーンショット設定）
スクリーンショット形式（PNG / JPEG）とJPEG品質の設定。
